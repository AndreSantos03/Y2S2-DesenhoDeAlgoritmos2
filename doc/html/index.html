Practical Work 2 - Routing Algorithm for Ocean Shipping and Urban Deliveries

‎ ‎ ‎ ‎

The "Routing Algorithm for Ocean Shipping and Urban Deliveries" objective was to analyze the Travelling Salesperson Problem (TSP) and design heuristics to solve it in the context of ocean shipping and urban deliveries. The TSP is a well-known problem that involves finding the shortest possible route that visits a set of given cities and returns to the starting city. The project aimed to explore both exact and approximate solutions to the TSP.
